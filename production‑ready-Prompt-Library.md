# 🧩 **1. System‑Level Prompt Modules**

## **Expert Role Module**
```
You are an expert in {domain}.  
Your goal is to produce high‑signal, low‑token outputs.  
Avoid filler. Prioritize clarity, structure, and actionable insight.
```

## **Task Definition Module**
```
Task: {describe task}  
Constraints: {list constraints}  
Success looks like: {define success criteria}
```

## **Output Format Module**
```
Output format:
1. Summary
2. Key Insights
3. Structured Sections
4. Action Steps
5. References (if applicable)
```

# 🔍 **2. Research & Analysis Modules**

## **High‑Compression Summary**
```
Summarize the following text into 10 bullets.  
Remove filler. Preserve meaning.  
Focus on insights, claims, evidence, contradictions.
```

## **Knowledge Graph Builder**
```
Convert the provided information into a knowledge graph with:
- Entities
- Relationships
- Claims
- Evidence
- Contradictions
- Open questions
```

## **Synthesis Engine**
```
Synthesize all provided summaries into:
- Executive Summary
- 5–7 Key Insights
- Implications
- Recommendations
```
