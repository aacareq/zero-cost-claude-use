## 🔧 **3. Optimization Playbook**

```markdown

## Context Packing
- Use semantic compression.
- Outline before detail.

- Split tasks into 4 passes: define, structure, fill, refine.

## External Memory
- Store summaries in Notes or Obsidian.
- Reinject compressed state.
```
